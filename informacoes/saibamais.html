HTML

HTML é a linguagem de marcação usada para estruturar páginas da web. Ele define os elementos básicos, como títulos, parágrafos, links, imagens e formulários. O HTML trabalha em conjunto com CSS e JavaScript para criar sites interativos e visuais.

CSS

CSS é responsável pela estilização das páginas HTML. Ele permite definir cores, fontes, espaçamentos e layouts, tornando as páginas mais atraentes e responsivas. Com CSS, também é possível criar animações e efeitos visuais.

JAVA SCRIPT

JavaScript é uma linguagem de programação que adiciona interatividade às páginas da web. Ele permite manipular elementos HTML e CSS dinamicamente, além de interagir com servidores, criar animações e desenvolver aplicativos web.

DOCKER

Docker é uma plataforma que permite empacotar aplicações e seus ambientes em contêineres. Isso facilita o desenvolvimento, a implantação e a escalabilidade de sistemas, garantindo que funcionem da mesma forma em diferentes ambientes.

FTP (File Transfer Protocol)

FTP é um protocolo usado para transferir arquivos entre um computador local e um servidor na internet. Ele é muito utilizado para fazer upload e download de arquivos de sites e sistemas web.
